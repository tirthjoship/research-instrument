# NIFTY 50 constituents (NSE, India) — snapshot as of 2025-12-08
# Source: Wikipedia "NIFTY 50" constituent table, cross-checked against
# Tickertape.in's NIFTY 50 stock list (both consulted 2026-07-16). NSE's own
# index-factsheet PDF (archives.nseindia.com/content/indices/ind_nifty50.pdf)
# was not machine-readable (image-based) at fetch time — could not be used
# as the primary source directly, hence the two independent secondary
# sources above instead of NSE's own document.
#
# CAVEATS (read before using this file as-is):
# - NSE rebalances NIFTY 50 semi-annually (March and September). This
#   snapshot is dated 2025-12-08; a March 2026 rebalance may not be
#   reflected. Re-verify against a live source before relying on this for
#   anything beyond development/testing.
# - Symbols WITHOUT the .NS suffix (added at runtime by
#   _get_backtest_universe, mirroring how tsx60.txt's bare symbols get .TO
#   added — see application/cli/_deps.py).
# - "M&M" (Mahindra & Mahindra), "BAJAJ-AUTO" (Bajaj Auto), and "TMPV"
#   (Tata Motors Passenger Vehicles, a 2025 demerger of Tata Motors into
#   separate passenger/commercial listings) — VERIFIED 2026-07-16 via a
#   real yfinance query (yf.Ticker("M&M.NS").info, etc.): "M&M.NS" ->
#   "Mahindra & Mahindra Limited", "BAJAJ-AUTO.NS" -> "Bajaj Auto Limited",
#   "TMPV.NS" -> "Tata Motors Passenger Vehicles Limited". No conversion
#   needed for any of the three (unlike TSX class shares, e.g. GIB.A ->
#   GIB-A.TO) — the bare symbol plus .NS resolves correctly as-is.
