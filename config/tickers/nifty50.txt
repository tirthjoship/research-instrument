# index-factsheet PDF (archives.nseindia.com/content/indices/ind_nifty50.pdf)

# sources above instead of NSE's own document.
#
# CAVEATS (read before using this file as-is):
# - NSE rebalances NIFTY 50 semi-annually (March and September). This
#   snapshot is dated 2025-12-08; a March 2026 rebalance may not be
#   reflected. Re-verify against a live source before relying on this for
#   anything beyond development/testing.
# - Symbols WITHOUT the .NS suffix (to be added at runtime, mirroring how
#   tsx60.txt's bare symbols get .TO added by _get_backtest_universe) —
#   suffix-handling code for India does not exist yet as of this file's
#   creation (see docs/superpowers/plans/2026-07-15-multi-market-screener-canada.md).
# - "M&M" (Mahindra & Mahindra) and "BAJAJ-AUTO" (Bajaj Auto) contain
#   non-alphanumeric characters. NOT YET VERIFIED against a live yfinance
#   query whether "M&M.NS" / "BAJAJ-AUTO.NS" are the correct literal
#   yfinance tickers, or whether they need conversion (the way TSX class
#   shares like GIB.A need dot-to-dash conversion to GIB-A.TO). Verify with
#   a real yfinance call before trusting these two specific tickers in
#   production.
# - "TMPV" (Tata Motors Passenger Vehicles) reflects a 2025 demerger of Tata
#   Motors into separate passenger-vehicle and commercial-vehicle listings.
#   Newer/less battle-tested ticker than the rest of this list — verify it
#   resolves correctly via yfinance before trusting it.
